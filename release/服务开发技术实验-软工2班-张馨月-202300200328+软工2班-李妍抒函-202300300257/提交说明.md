# 课程实验提交说明

本目录用于提交《服务开发技术》课程实验最终材料。

## 交付内容

- `documents/服务开发技术课程实验报告.word`：课程实验报告。
- `documents/服务开发技术课程实验报告.pdf`：课程实验报告。
- `documents/api/`：Postman Collection 与接口说明。
- `documents/screenshot/`：系统页面与 Postman 验证截图。
- `service/weekend-go-backend-0.0.1-SNAPSHOT.war`：最终后端服务 War 包。
- `database/schema.sql`：MySQL 建表脚本。
- `database/dev_seed.sql`：本地演示数据脚本。


## 运行说明

后端基于 Spring Boot、Java 17、MySQL。部署前请准备 MySQL 数据库，并按需执行 `database/schema.sql` 和 `database/dev_seed.sql`。

War 包可作为 Spring Boot 可执行 War 运行：

```bash
java -jar service/weekend-go-backend-0.0.1-SNAPSHOT.war
```

如部署到外部 Servlet 容器，请确认容器兼容 Jakarta Servlet / Spring Boot 3。

本提交包已排除 `.git`、`.codex`、`.local`、`.worktrees`、`node_modules`、Maven/前端构建中间目录、本地日志、Postman 安装程序、临时截图和其他开发缓存。
